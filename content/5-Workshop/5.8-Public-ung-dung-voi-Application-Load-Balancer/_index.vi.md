## 5.8. Public ứng dụng với Application Load Balancer

Phần này public EC2 backend đang nằm trong private subnet thông qua Application Load Balancer. Các bước gồm tạo Target Group, đăng ký backend, cấu hình HTTP listener và kiểm tra truy cập bằng ALB DNS name.
